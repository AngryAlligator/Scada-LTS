Hoy en día, se soportan 3 tipos de datos: Punto simple, punto doble Medición de puntos estandarizados.

Commandos

Para enviar comandos, puede especificar un offset, que se añadirá a la dirección del objeto cuando se envíe el comando. Por ejemplo: cuando se escribe la dirección 2 de un objeto con el offset 2000, se enviará el comando para oponerse con la dirección 2002.
